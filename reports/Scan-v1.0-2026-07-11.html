4. What to steal — per vendor

IQGeo:
  • Capture-time validation loop (NetLux): photo analyzed in <2 s as taken -> technician fixes on-site -> auto-acceptance triggers contractor payment. Metric to copy: First Time Right rate.
  • Their design automation (Comsof) is deterministic optimization: polygon in -> lowest-cost layout + BOM out. Our LLM design must show what theirs can’t: reasoning, alternatives, dialogue on the map.
  • M&A pattern (Comsof 2022, Deepomatic 2025): they buy capabilities; nothing LLM-native shipped yet.
  • New this period: <2 s validation sets UX bar for mobile capture-time trust.
Netcracker:
  • Agent catalog + per-agent KPI/consumption dashboard = management view + billing meter in one artifact.
  • Trust Gateway pattern: log intent, retrieved data, and rationale by default in the orchestrator.
  • MCP + A2A support = direct validation of our inter-agent protocol work.
  • New this period: AI Excellence Award validates agentic platform as a blueprint.
ServiceNow:
  • Embedded generation in the workspace: record-history summary + drafted notes next to the record (~80% faster resolution notes). Transfers 1:1 to our work orders/tickets.
  • Agent Advisor concept: rank which of our agents pays off first on the customer’s own data — doubles as a sales tool.
  • Tiering by AI maturity (Foundation / Advanced / Prime) — ready framing for Agent packaging.
Salesforce:
  • Productize our rules+LLM as "hybrid reasoning": deterministic where it must be, generative where it helps. Kills the reliability objection.
  • HITL standard: approval status on AI-created/updated records + per-field "AI-filled" markers.

5. Standard UX kit (implement, don't invent)

  1. One visible assistant; orchestrator routes to hidden specialized agents.
  2. Context-aware quick actions on record/grid/map: "Ask", "Summarize", prefilled prompts opening a chat with context attached.
  3. Agent transparency stream: agent narrates plan, user can stop/redirect mid-run.
  4. HITL statuses: AI fills -> per-field AI markers -> pending approval -> approve/revert.
  5. Embedded generation next to the record, not in a separate tool.
  6. Agent catalog + per-agent KPI/consumption metering.
  7. Capture-time mobile validation (photo checked as taken, <2 s response).
  8. Consumption pricing per agent action: simple assist cheap, multi-step xN.
